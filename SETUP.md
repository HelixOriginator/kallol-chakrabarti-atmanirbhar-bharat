# 🚀 Quick Setup Guide

## How to Create This Repository on GitHub

### Step 1: Create New Repository
1. Go to [github.com/new](https://github.com/new)
2. **Repository name:** `kallol-chakrabarti-atmanirbhar-bharat`
3. **Description:** `Citizen-built national knowledge infrastructure — Independent research contributions to Atmanirbhar Bharat & Viksit Bharat 2047 by Kallol Chakrabarti`
4. Set to **Public**
5. Click **Create repository**

### Step 2: Upload All Files
Upload every file from this package maintaining the folder structure exactly as provided.

### Step 3: Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Source: **Deploy from a branch** → `main` → `/ (root)`
3. Click **Save**

Your site will be live at:
`https://helixoriginator.github.io/kallol-chakrabarti-atmanirbhar-bharat/`

### Step 4: Add Repository Topics
Go to repository → ⚙️ (gear icon next to About) → Add topics:
```
atmanirbhar-bharat viksit-bharat-2047 india ai-governance policy-research
independent-research nano-banana patent-democracy civic-technology
```

### Step 5: Update Repository Description
Set the website URL to your GitHub Pages URL in the About section.
